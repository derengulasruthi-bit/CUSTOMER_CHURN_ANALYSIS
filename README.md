# Customer Churn Analysis & Prediction

## Project Objective
Analyze customer behavior and predict customer churn using data analytics and machine learning.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jira

## Key Tasks Performed
- Data Cleaning
- Exploratory Data Analysis
- Correlation Analysis
- Data Visualization
- Logistic Regression Model
- Churn Prediction

## Key Insights
- Young customers showed higher churn behavior.
- Low balance customers were more likely to churn.
- Inactive customers had higher churn probability.
- The Logistic Regression model achieved approximately 80% accuracy.

## Business Recommendations
- Improve engagement for inactive customers.
- Provide loyalty benefits for younger customers.
- Monitor low-balance customer activity.

## Project Outcome
Successfully built an end-to-end customer churn analysis and prediction project using Python and machine learning techniques.
